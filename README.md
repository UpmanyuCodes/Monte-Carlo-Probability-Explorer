# Monte Carlo Probability Explorer

Monte Carlo Probability Explorer is a Python-based project that uses simulation techniques to explore fundamental concepts in probability and statistics. Through repeated random sampling, the project demonstrates how theoretical probabilities emerge from experimental results and provides insight into several well-known probabilistic phenomena.

## Simulations Included

- **Coin Toss Simulation** – Illustrates the Law of Large Numbers by showing the convergence of experimental probability toward 0.5.
- **Dice Roll Simulation** – Demonstrates how the frequency of each outcome approaches the theoretical probability of 1/6.
- **Monte Carlo Estimation of π** – Estimates the value of π using randomly generated points.
- **Birthday Paradox Simulation** – Explores the probability of shared birthdays within a group.
- **Monty Hall Problem Simulation** – Compares switching and staying strategies, confirming that switching wins approximately 2/3 of the time.

## Technologies Used

- Python
- Matplotlib
- Random Module

## Concepts Demonstrated

- Monte Carlo Simulation
- Law of Large Numbers
- Experimental vs Theoretical Probability
- Statistical Estimation
- Conditional Probability

## Objective

The goal of this project is to make probability concepts more intuitive by visualizing them through computational experiments and simulations.

## Key Takeaway

> Repeated random experiments reveal the underlying patterns of probability, helping bridge the gap between theory and observation.
